> 周期：第 7-9 天
> 目标：`TaobaoSpider.search(keyword, page)`，统一输出 `SearchResult`。

## 任务清单

- [ ] 3.1 安装 DrissionPage：`pip install DrissionPage`
- [ ] 3.2 编写 `scripts/taobao_login.py`：
  - 启动 ChromiumPage 打开 `https://login.taobao.com`
  - 用户手动扫码登录
  - 提示用户回车后导出 cookies 到 `state/taobao_cookies.json`
- [ ] 3.3 编写 `spiders/taobao.py`：
  - 类 `TaobaoSpider(BaseSpider)`，platform=Platform.TAOBAO
  - `__init__`：从 state 加载 cookies 注入到 ChromiumPage
- [ ] 3.4 `search(keyword, page=1)`：访问 `https://s.taobao.com/search?q=xxx&s={(page-1)*48}`
- [ ] 3.5 解析 `g_page_config` JSON 提取 `mods.itemlist.data.auctions[]`
- [ ] 3.6 字段映射：`nid → item_id`，`raw_title → title`，`view_price → current_price`，`view_sales → sales`，`item_loc → location`，`nick → shop_name`
- [ ] 3.7 滑块检测：DOM 含 `#nc_1_wrapper` 或跳转 `login.taobao.com` → 抛 `LoginRequiredError`，提示用户重新登录
- [ ] 3.8 `dump_debug_artifact` 在解析失败时保存完整 HTML 与截图
- [ ] 3.9 `tests/test_taobao.py` 集成测试

## 已知坑

- **必须登录态**，否则搜索页直接返回登录页
- 滑块出现概率与请求频率高度相关，**默认 `min_interval=3.0` 秒**
- `g_page_config` 在 `<script>` 里，正则 `g_page_config\s*=\s*(\{.*?\});` 提取（注意贪婪/非贪婪）
- 无图模式 / 海外淘宝 / 天猫商品字段不同，先只支持普通搜索结果
- DrissionPage **不支持纯无头**（淘宝会识别），必须 headed 或 xvfb

## 验收标准
- `TaobaoSpider().search("RTX 4090", page=1)` 返回 ≥40 条
- 滑块出现时抛出 `LoginRequiredError`，并在日志记录截图路径
- 删除 cookie 后调用立即抛 `LoginRequiredError`
