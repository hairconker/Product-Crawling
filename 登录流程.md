# 四平台登录 · 扫码自动保存（推荐）

> **推荐方案**：跑一个脚本 → 自动弹出浏览器 → 扫码 → 自动落盘 cookie → 结束。
> 不用手动 F12 复制。

---

## 一、一次性准备（仅第一次）

### 1.1 确认 pip 可用

在 PowerShell 里：

```powershell
cd E:\autoPCBulid
python -m pip --version
```

如果提示找不到 pip：

```powershell
python -m ensurepip --upgrade
# 如果还不行：
python -m pip install --upgrade pip
```

### 1.2 装 Playwright（一次性，约 150MB 浏览器）

```powershell
pip install playwright
playwright install chromium
```

---

## 二、扫码登录（每次 cookie 过期再跑一次）

### 2.1 一次性登录 4 个平台

```powershell
python scripts\login_helper.py all
```

脚本会依次弹出浏览器：京东 → 闲鱼 → 淘宝（拼多多已移除）。
每弹一次的操作：

1. 在浏览器里**扫码 / 输账号密码**完成登录
2. 登录成功后，**回到此 PowerShell 终端**，按 **回车键**
3. 脚本立刻保存 cookie 并关闭浏览器，自动进入下一个平台
4. 如果某个平台不想登，直接在终端输入 `s` 后回车跳过

### 2.2 只登录某一个平台

```powershell
python scripts\login_helper.py jd       # 京东
python scripts\login_helper.py xianyu   # 闲鱼
python scripts\login_helper.py taobao   # 淘宝
```

### 2.3 完成后会生成

```
state/
├── jd_state.json         # Playwright 完整登录态（以后自动登录用）
├── jd_cookie.txt         # cookie 字符串（run_cpu_crawl.py 用）
├── xianyu_state.json
├── xianyu_cookie.txt
├── taobao_state.json
├── taobao_cookie.txt
├── taobao_state.json
└── taobao_cookie.txt
# 拼多多已移除
```

## 三、开始爬取

```powershell
python run_cpu_crawl.py "i5-12400F"
```

或单独只跑某个平台：

```powershell
python run_cpu_crawl.py "i9-14900K" --only jd
python run_cpu_crawl.py "R7 7800X3D" --only taobao
```

---

## 淘宝滑块 & 账号保护（重要）

从这版起：
- **爬虫自动降速**：滚动间隔 2.2s、翻页 4.5s、跨平台 6s —— 保护账号不被封
- **淘宝检测到滑块时不再立即退出**：会自动等待最多 **5 分钟**你手动拖动滑块。条件：**必须用 `--headed` 运行**（无头模式无法操作）

```powershell
python run_cpu_crawl_pw.py "i5-12400F" --pages 3 --headed
```

跑的时候看到浏览器跳出滑块 → 你手动拖 → 爬虫自动继续。终端会打印：`[taobao] 检测到滑块/验证码，请在浏览器中手动完成`。

---

## 四、登录等待超时

脚本默认等 **10 分钟**回车。如果超时，扫码没完成就按回车也没关系——但脚本会认为你没登录成功，保存的 cookie 可能是未登录态，之后爬取会报 `LoginRequiredError`。

**正确做法**：扫码 / 登录完成后再回到终端按回车。

## 五、手动备选方案（不装 Playwright）

如果你不想装 Playwright，也可以手动从浏览器 F12 复制 Cookie：

<details>
<summary>点开查看手动步骤</summary>

每个平台步骤一致：

1. Chrome 访问对应登录页并登录
2. 打开搜索页（JD: search.jd.com/Search?keyword=cpu；淘宝: s.taobao.com/search?q=cpu；闲鱼: goofish.com/search?q=cpu）
3. F12 → Network → 刷新 → 点第一条主文档请求 → Headers → Request Headers → 复制 `Cookie:` 整行
4. 粘贴到 `E:\autoPCBulid\state\{platform}_cookie.txt`（platform = jd / xianyu / taobao）

</details>

## 六、Cookie 安全提示

- `state/` 已在 `.gitignore`，不会被 git 提交
- Cookie 等同账号登录态，**不要发给他人**
- 建议爬虫用小号，别用主账号

---

## 七、出错时排查

日志路径：
- `logs/crawl_YYYY-MM-DD.log` —— 全量
- `logs/error.log` —— 仅错误
- `logs/debug/{platform}_*.html` —— 出错时自动落盘的响应，方便你截图给我排查

任何平台跑失败不会影响其他平台，最终 `logs/report_*.json` 会汇总全部结果。
